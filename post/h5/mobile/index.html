移动端样式1px边框实现

1.伪类实现

.borderline{
    width: 100%;
    height: 0;
    position: relative;
    border:none;
}
.borderline:after{
    content: '';
    position: absolute;
    z-index: 999;
    left: 0;
    /* background: rgb(230,230,230); */
    background: rgb(224,224,224);
    width: 100%;
    height: 1px;
    -webkit-transform: scaleY(0.5);
            transform: scaleY(0.5);
    -webkit-transform-origin: 0 0;
            transform-origin: 0 0;
}
